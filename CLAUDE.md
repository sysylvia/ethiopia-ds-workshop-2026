# Ethiopia DS Workshop 2026

## Repository Purpose
WISE Project training workshop website: Data Sciences for Modeling and Health Technology Assessment.
February 2-5, 2026 | Addis Ababa University, Ethiopia

## Creating Slides

Use the `lecture-slides` skill when creating new slide decks:
```
/lecture-slides {topic}
```

### Slide Format Requirements
- Quarto reveal.js format
- Use custom theme: `[default, ../style/workshop.scss]`
- Include: slide-number, chalkboard, transition: fade
- Footer: "WISE Workshop | Data Sciences for Modeling & HTA"
- Pattern from existing Day 2 slides (`day-2/slides-ml-intro.qmd`)

### Directory Structure
- Slides go in: `day-{N}/slides-{topic}.qmd`
- Notebooks go in: `day-{N}/notebook-{topic}.qmd`
