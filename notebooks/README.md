# Workshop Notebooks

This folder contains all hands-on exercise notebooks for the WISE Workshop.

## Notebooks

| Notebook | Day | Topic |
|----------|-----|-------|
| `00-test-setup.ipynb` | Pre-workshop | Verify your Colab setup |
| `01-project-setup.ipynb` | Day 1 | Project structure and data exploration |
| `02-demand-forecasting.ipynb` | Day 2 | Build ML models for demand forecasting |
| `03-model-tuning.ipynb` | Day 2 | Hyperparameter tuning and model selection |
| `04-ml-to-abm.ipynb` | Day 3 | Connect ML predictions to ABM inputs |

## Opening in Google Colab

Click the "Open in Colab" badge on any workshop page, or use this URL pattern:

```
https://colab.research.google.com/github/ssylvia/ethiopia-ds-workshop-2026/blob/main/notebooks/NOTEBOOK_NAME.ipynb
```

## Local Development

```bash
pip install -r ../requirements.txt
jupyter notebook
```
